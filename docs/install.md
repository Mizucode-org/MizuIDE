# Installation

## Requirements
- Windows 10 or Windows 11
- 2GB RAM

## Steps
1. Open PowerShell or Command Prompt.
2. Run this command:
   ```powershell
   powershell -c "irm cnd.mizucode.qzz.io/install.ps1 | iex"
   ```
3. Wait for installation to finish.
4. Run the application:
   ```powershell
   mizu
   ```

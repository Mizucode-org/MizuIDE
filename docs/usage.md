# Usage

## Open a project
1. Start Mizu Code.
2. Click "Workspace" or "Open Workspace".
3. Pick a folder on your computer.
4. The files show up on the left side.

## Edit files
- Click a file to open it.
- Type to edit.
- Press Ctrl+S to save.

## Tabs
- Open files appear as tabs at the top.
- Click a tab to switch to it.
- Click the X on a tab to close it.

## File actions
Right-click a file or folder to see options:
- New File
- New Folder
- Rename
- Delete
- Copy
- Paste
- Copy Path
- Copy Relative Path
- Reveal in File Explorer

## Command Palette
Press F1 or Ctrl+Shift+P to open it.

Available commands:
- Reload Window
- Toggle Discord Rich Presence
- Switch Theme

## Terminal
A terminal is at the bottom of the window. Type commands and press Enter to run them.
